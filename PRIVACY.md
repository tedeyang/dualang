# 隐私政策

**X 光速翻译**（以下简称"本扩展"）尊重用户隐私。本政策说明本扩展收集、使用和存储的信息。

**最后更新日期：2026-04-20**

## 我们收集的信息

本扩展**不收集**任何个人身份信息。

### 本地存储的数据

以下数据仅存储在您的浏览器本地（`chrome.storage.sync` / `chrome.storage.local`），**不会上传到任何服务器**：

- **API 设置**：您配置的 API 地址、API Key、模型名称、目标语言等翻译参数
- **用户偏好**：是否自动翻译、展示模式、是否启用兜底 API 等开关状态
- **翻译缓存**：已翻译推文的文本和结果，用于减少重复 API 调用
- **统计数据**：请求次数、缓存命中率、最近错误（仅用于本地展示）

### API 调用

本扩展需要调用您配置的第三方翻译 API（如 SiliconFlow、Moonshot 等）：

- 仅发送**推文文本内容**到您指定的 API 端点
- API Key 仅用于向该服务商验证身份，不会发送到其他服务器
- 翻译响应结果仅缓存在本地浏览器

## 我们不会做的事

- 不会追踪您的浏览历史
- 不会收集您的 X.com / Twitter 账号信息
- 不会将您的推文内容或 API Key 发送到除您配置的服务商以外的任何服务器
- 不会使用 Cookie 或第三方分析工具

## 数据安全

- API Key 存储在浏览器本地存储中，受浏览器沙箱保护
- 建议通过 `config.json` 文件配置 API Key，该文件不会被纳入版本控制
- 请勿与他人分享包含您 API Key 的 `config.json` 文件

## 权限说明

本扩展申请的浏览器权限用途如下：

| 权限 | 用途 |
|---|---|
| `storage` | 保存用户设置和翻译缓存 |
| `activeTab` | 在 X.com 页面注入翻译内容 |
| `contextMenus` | 提供右键菜单快速开关翻译 |
| `tabs` | 向已打开的 X.com 标签页广播设置变更 |
| `alarms` | 保持 Service Worker 存活，避免长滚动时翻译中断 |
