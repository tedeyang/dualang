# X 光速翻译

**刷 X 时，译文应该像空气一样不被察觉。**

专为 **X.com / Twitter** 打造的沉浸式翻译插件。它不做弹窗面板，而是让译文直接长在时间线上——滚动到视野内的推文自动译好，点击「译」按钮毫秒出结果，展开 Show more、刷新页面、切换网络，体验都丝般顺滑。

**为什么快？** 视窗检测 + 本地缓存，大部分翻译零网络等待。**为什么稳？** 主 API 与兜底 API 并发赛跑，谁先到用谁，主线路卡顿自动无缝切换。**为什么准？** 智能难词预筛 + 段落级词典融合 + 逐行双语对照，专业术语不再望文生义。

支持 SiliconFlow（GLM-4-9B / Qwen 免费模型）、Moonshot Kimi、阿里通义千问等多家模型，API Key 仅保存在本地浏览器，隐私零妥协。

---

## 功能一览

- 🎯 **专为 X.com 深度优化**：自动识别推文正文，一键翻译，完美融入暗黑/亮色主题
- ⚡ **视窗即译**：滚动到视野内的推文自动触发翻译，减少无效调用
- 🔘 **点击秒回**：手动模式下每条推文右下角出现「译」按钮，点击即译
- 🏇 **多模型赛马**：主 API 与兜底 API 并发请求，先到先用；主线路故障自动无缝切换
- 🎨 **四种展示模式**：保留原文 / 仅译文 / 段落对照 / 整体对照
- 📖 **智能字典融合**：自动标注生僻词音标与释义，段落级精准融合
- ✨ **逐行双语对照**：长推按原文行结构逐行拼接译文，阅读节奏不被打乱
- 🔄 **流式输出**：长文边译边看，无需等待全文返回
- 🌐 **多语言支持**：简中、繁中、英、日、韩、法、德、西、俄等语言互译
- 🔒 **隐私优先**：API Key 与设置仅存储在本地浏览器，不经过任何第三方服务器

## 安装

### 方式一：加载已解压的扩展程序（推荐，随时更新）

1. 下载本项目，进入 `dist/x-light-translate/` 文件夹（已包含全部运行文件）
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `dist/x-light-translate/` 文件夹
6. 安装成功后，浏览器右上角出现「X 光速翻译」图标

### 方式二：ZIP 包安装

1. 下载 `dist/x-light-translate.zip`
2. 解压到任意文件夹
3. 按方式一的步骤 2~5 加载该文件夹

## 配置插件

点击浏览器右上角的「X 光速翻译」图标，弹出设置面板：

### 第一步：获取 API Key（免费）

推荐 **SiliconFlow**（默认），GLM-4-9B 和 Qwen 模型均可免费使用：

1. 前往 [SiliconFlow 官网](https://cloud.siliconflow.cn/) 注册账号
2. 进入「API 密钥」页面，创建新密钥（格式如 `sk-...`）
3. 复制密钥，粘贴到插件设置的「API Key」框中

> 默认 API 地址已填好 `https://api.siliconflow.cn/v1`，模型为 `THUDM/GLM-4-9B-0414`，无需改动即可使用。

### 第二步：选择目标语言

在「目标语言」下拉菜单中选择你希望看到的翻译语言，默认是「简体中文」。

### 第三步：开启自动翻译（可选）

勾选「自动翻译推文」后，滚动页面时进入视野的推文会自动翻译，无需手动点击。

> 自动翻译会消耗 API 调用额度，请根据需求开启。每条推文只翻译一次，本地缓存有效期内不再重复请求。

### 第四步：配置兜底 API（可选，推荐）

在主 API 设置下方，可以勾选「启用兜底 API」并填入第二组 API 地址和 Key。当主 API 响应慢或故障时，插件会自动同时向两组 API 发请求，谁先到用谁，翻译不中断。

> 建议主 API 用 SiliconFlow（免费），兜底 API 用 Moonshot Kimi 或其他备用渠道。

## 使用说明

### 基础翻译

- **自动模式**：开启后，滚动页面时推文自动翻译，右下角显示模型图标表示翻译成功
- **手动模式**：关闭自动翻译后，每条未翻译的推文右下角会出现「译」按钮，点击即译
- **重试**：若翻译结果明显异常（如过短、语言错误），插件会自动重试一次；若仍失败，可点击译文区域手动重试

### 展示模式

点击页面右侧浮球面板上的「对照风格」按钮，切换以下四种显示方式：

| 模式 | 效果 |
|------|------|
| **保留原文**（默认） | 译文显示在原文下方，适合快速浏览 |
| **仅译文** | 隐藏原文，只显示译文，界面最简洁 |
| **段落对照** | 每段原文与译文逐段并排，适合精读 |
| **整体对照** | 整篇原文与整篇译文上下对照，适合长文 |

> 切换模式后，已翻译的推文不会立即变化，新加载或重新翻译的推文会使用新模式。

### 逐行融合

在浮球面板中开启「逐行融合」开关后，多行推文会按原文的行结构逐行展示译文，每行原文与译文之间用细线分隔，阅读节奏与原文保持一致。

> 仅在「保留原文」和「整体对照」模式下生效。单行推文不会触发融合。

### 智能字典

在浮球面板中开启「智能字典」开关后，英文原文中的生僻词、俚语、专业术语会自动标注音标和简短释义，悬浮或点击即可查看。

> 仅在英文原文且非「仅译文」模式下生效。字典标注不影响原文排版。

### 长文精翻（浮球）

当遇到 X Articles 长文（4000 字以上）时，插件不会自动翻译，而是在页面右侧出现一个「🌐」浮球：

- 点击浮球开始精翻，按段落逐段翻译，带进度环显示
- 翻译过程中可 hover 浮球查看进度，或点击「取消」停止
- 精翻完成后，译文按原文章节结构插入，保留原文中的图片、视频、链接
- 浮球支持上下拖动调整位置，位置会自动记住

### 快捷设置面板（浮球）

页面右侧的浮球不只是长文入口，还是常驻快捷设置面板：

- **开关翻译**：一键开启/关闭全部翻译
- **切换模式**：快速切换四种展示模式
- **逐行融合**：开关逐行对照
- **智能字典**：开关难词标注
- **切换模型**：在已配置的模型间快速切换，显示当前模型平均响应速度

## 常见问题

**Q: 翻译速度很慢？**  
A: 首次翻译需要网络请求，后续同一内容从本地缓存读取，基本零延迟。可在浮球面板切换响应更快的模型，或开启「启用兜底 API」加速。

**Q: 自动翻译消耗很多额度？**  
A: 插件有双层缓存，同一条推文只翻译一次。如果担心额度，关闭「自动翻译」，改为手动点击「译」按钮。

**Q: 译文里图片/视频不见了？**  
A: 长文精翻会保留原文中的图片和视频。普通推文的媒体元素在「保留原文」模式下仍显示在原文区域，不受影响。

**Q: 部分推文没有翻译按钮？**  
A: 纯英文以外的推文（如已有中文、日文、韩文）会被自动跳过。URL 链接推文、空白推文也会被跳过。

**Q: 如何重置统计数据？**  
A: 打开插件 popup → Stats 标签页 → 点击「重置统计」按钮。
